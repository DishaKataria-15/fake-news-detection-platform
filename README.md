# 📰 Fake News Detection Platform

A Machine Learning-powered platform designed to detect and analyze fake news articles using real-time verification and intelligent text classification techniques.

---

# 🚀 Project Overview

This project focuses on identifying fake and misleading news content using Machine Learning and Natural Language Processing (NLP).

The platform integrates:
- Frontend Interface
- Backend Processing
- Machine Learning Models

to provide real-time fake news analysis and verification.

# ⚙️ Working Process

1. User enters news/article text
2. Backend processes the input
3. NLP techniques clean and analyze the text
4. Machine Learning model predicts:
   - Fake News
   - Real News
5. Result displayed through frontend interface

# ▶️ How to Run the Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/DishaKataria-15/fake-news-detection-platform.git
```

---

## 2️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Start Backend Server

```bash
python app.py
```

OR

```bash
python manage.py runserver
```

(depending on backend setup)

---

## 4️⃣ Open Frontend

Run frontend locally or open browser interface.
